# Sion Kim AI-Driven Next.js Portfolio

경력직 웹퍼블리셔 김시온의 2026년형 AI 활용 포트폴리오입니다.

## 특징

- Next.js App Router 기반
- React 컴포넌트 분리
- `data/profile.js`에서 경력/프로젝트 내용 관리
- GitHub Pages 정적 배포용 설정 포함
- GitHub Actions 자동 배포 workflow 포함

## 로컬 실행

```bash
npm install
npm run dev
```

## GitHub Pages 배포

Settings → Pages → Source를 `GitHub Actions`로 설정하세요.

main 브랜치에 push하면 자동 배포됩니다.
